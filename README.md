# FHN-SOM-Clustering

This repository contains the figures, supplementary material, and source code associated with the paper:

"Behavior-Oriented Clustering of Nonlinear Dynamical Systems Using Self-Organizing Maps"

## Repository Structure

- Figures/
  - Cluster_1
  - Cluster_2
  - Cluster_3
  - Cluster_4
  - Cluster_5
  - Cluster_6

Each cluster folder contains:

- Time-series plots
- Phase portraits
- 3D trajectories
- Return maps
- Poincaré maps
- Chaotic attractors

The repository also includes the MATLAB, R, and Transformer implementations used in this study.
